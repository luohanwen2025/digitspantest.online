# 数字广度测试分享功能 PRD
**Product Requirements Document**

---

## 📋 **1. 产品概述**

### **1.1 项目背景**
- **项目名称**: 数字广度测试分享功能
- **产品目标**: 增强网站自传播能力，通过用户分享获得新用户
- **目标用户**: 完成测试的用户（18-45岁，对认知测试和个人提升感兴趣）
- **业务价值**: 提升品牌曝光、增加自然流量、提高用户粘性

### **1.2 核心价值主张**
> "让每一次测试都成为分享的骄傲"
- 用户完成测试后可生成精美分享卡片
- 支持一键分享到主流社交平台
- 通过数据可视化展示个人成果
- 激发用户分享欲望和平台互动

### **1.3 成功指标**
- 分享卡片生成率：≥60%（完成测试后生成）
- 分享点击率：≥25%（分享后被点击）
- 分享转化率：≥8%（点击后注册/测试）
- 社交媒体讨论度：持续增长

### **2.2 功能详述**

#### **功能1：分享卡片生成器**
**功能描述**: 根据用户测试结果自动生成精美的分享卡片

**输入数据**:
- 测试分数（0-1050）
- 错误率百分比
- 测试等级（初学者/良好/优秀/卓越）
- 百分位排名
- 完成时间
- 改进建议（3-5条）

**输出内容**:
- 主视觉区域：成绩可视化图表
- 分数展示：当前分数+等级徽章
- 排名信息：超过X%的用户
- 改进建议：个性化建议文本
- 品牌元素：网站Logo+URL
- 互动按钮："挑战朋友"行动召唤

**技术要求**:
- 卡片尺寸：1200x630px（Open Graph标准）
- 格式：PNG图片
- 清晰度：2x Retina支持
- 加载时间：<2秒

#### **功能2：社交平台分享**
**功能描述**: 支持一键分享到主流社交平台

**支持平台**:
1. **X (Twitter)** - 主要平台
   - 分享内容：卡片图片+文案+短链
   - 字符限制：280字符
   - 特性：支持话题标签

2. **Reddit** - 社区平台
   - 分享内容：卡片图片+详细文案
   - 目标subreddit：r/brain, r/Memory, r/puzzles
   - 特性：支持多图

3. **Facebook** - 社交平台
   - 分享内容：卡片图片+文案
   - 特性：支持缩略图预览

4. **复制链接/下载** - 备用方案
   - 复制分享链接
   - 下载卡片图片到本地

**技术实现**:
- Web Share API（原生分享）
- 自定义分享弹窗（兼容性）
- 平台特定URL scheme

#### **功能3：预设文案库**
**功能描述**: 提供8种不同风格的预设分享文案

**文案模板**:

1. **🏆 挑战型**
   > "我刚完成了数字广度测试，得分XXX分！我的工作记忆能力打败了XX%的用户。敢来挑战我吗？\n\n👉 测试链接：{url} #数字广度测试 #工作记忆"

2. **📊 统计型**
   > "刚做完认知测试，在数字广度任务中得了XXX分，超过了XX%的参与者！我的大脑记忆力居然这么强？\n\n🧠 测测你的：{url} #认知测试"

3. **🧠 专业型**
   > "工作记忆容量测试结果：XXX分（等级：XXX）\n\n这是认知心理学中的经典测试，反映大脑的注意力控制和信息处理能力。科学又有趣！\n\n📈 你的结果：{url}"

4. **🎯 成长型**
   > "从XXX分提升到XXX分！经过练习，数字广度测试成绩稳步上升。工作记忆真的可以训练！\n\n💪 开始你的训练：{url} #脑力训练"

5. **😂 幽默型**
   > "我的大脑：记住7位数字？小事一桩！\n\n实际测试：我能记住XXX位数字（骄傲脸）\n\n哈哈，你敢来试试吗？{url}"

6. **🔥 炫耀型**
   > "又破纪录啦！XXX分达成！🎉\n\n在数字广度测试中创造个人最好成绩，超越XX%的用户！我就是记忆力王者！\n\n👑 挑战我：{url}"

**文案特点**:
- 包含行动召唤（CTA）
- 添加相关话题标签
- 包含分享链接（短链）
- 符合各平台字符限制
- 情感化表达，提高分享欲望

#### **功能4：本地保存功能**
**功能描述**: 用户可下载分享卡片到本地

**技术实现**:
- HTML2Canvas截取卡片
- FileSaver.js触发下载
- 压缩优化文件大小（<500KB）

| **FileSaver.js** | 文件下载 | 2.0.5 | 简单易用 |

### **4.2 核心模块实现**

#### **模块1：ShareCardGenerator**
**职责**: 生成分享卡片图片

```javascript
class ShareCardGenerator {
  constructor(data) {
    this.data = data;
    this.canvas = document.createElement('canvas');
    this.ctx = this.canvas.getContext('2d');
  }

  generate() {
    // 1. 设置画布尺寸
    this.setCanvasSize(1200, 630);

    // 2. 绘制背景
    this.drawBackground();

    // 3. 绘制品牌元素
    this.drawBrand();

    // 4. 绘制成绩图表
    this.drawChart();

    // 5. 绘制分数区域
    this.drawScore();

    // 6. 绘制建议
    this.drawSuggestions();

    // 7. 绘制CTA按钮
    this.drawButtons();

    // 8. 导出图片
    return this.canvas.toDataURL('image/png');
  }
}
```

#### **模块2：SocialShare**
**职责**: 处理社交平台分享

```javascript
class SocialShare {
  static async shareToTwitter(cardData) {
    const text = this.getTemplateText('challenge', cardData);
    const url = `https://twitter.com/intent/tweet?text=${encodeURIComponent(text)}`;
    window.open(url, '_blank');
  }

  static async shareToReddit(cardData) {
    const text = this.getTemplateText('educational', cardData);
    const url = `https://reddit.com/submit?url=${encodeURIComponent(cardData.shareUrl)}&title=${encodeURIComponent(text)}`;
    window.open(url, '_blank');
  }

  static async shareToFacebook(cardData) {
    const url = `https://www.facebook.com/sharer/sharer.php?u=${encodeURIComponent(cardData.shareUrl)}`;
    window.open(url, '_blank');
  }
}
```

### **4.3 性能优化**

#### **4.3.1 卡片生成优化**
- 图片缓存：相同成绩复用卡片
- 懒加载：点击分享时才生成
- 压缩：Canvas图片压缩至80%质量
- 预加载：预渲染常见分数卡片

#### **4.3.2 分享体验优化**
- 短链服务：减少分享链接长度
- 预取预览：提前获取分享页面
- 智能推荐：根据用户选择推荐文案
- 批量操作：一键分享到多个平台
